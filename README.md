This repo is a project for building a Node App Api.
This repo is forked from jonas Schmedtmann's [complete-node-bootcamp](https://github.com/jonasschmedtmann/complete-node-bootcamp)
You should have node.js, VS code downloaded, and github linked.

Steps to run the code:

1. go to VS terminal, 
2. type "cd 1-node-farm
3. type "cd starter"
4. type "node .\index.js" to have the port running
